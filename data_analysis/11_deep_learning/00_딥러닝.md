# 데이터 분석을 위한 딥러닝 기초
## 딥러닝 개념 및 흐름
### 딥러닝(Deep Learning)
데이터를 반복 학습하며 패턴을 찾는 인공지능 기술

-> 영상, 음성, 텍스트 등 다양한 데이터에 활용됨

- 딥러닝과 머신러닝 차이
    - 머신러닝은 사람이 특징을 설계해야 함
    - 딥러닝은 특징 추출까지 모델이 자동으로 수행함
    - 모두 학습 기반이지만 처리 방식에 차이가 있음
